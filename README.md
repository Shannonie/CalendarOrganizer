# TaskOrganizationAPI
A data-driven WPF application with a tech-user interface.
- Techniques:
  - MVVM
  - Dependency Injection
  - Delegate Commands
  - Asynchronous Programming
  - EF code first
  - Optimistic Concurrency

# Libraries
- Entity Framework 
- AutoFac
- Prism
- Mahapps.Metro
- Extended.WPF.Toolkit
